# Движение остатков — МойСклад

Веб-приложение для отслеживания движения товарных остатков с интеграцией МойСклад.

## Возможности

- Авторизация по логину и паролю
- Загрузка данных из МойСклад (оприходования и списания)
- Таблица всех товаров с группировкой и раскрытием групп
- Разбивка по размерам одежды (XS / S / M / L / XL / XXL)
- Фильтр по группе и поиск по названию
- График движения по дням
- Экспорт в Excel (два листа: товары и сводка по группам)
- Демо-режим без токена

## Технологии

Одностраничное приложение на чистом HTML/CSS/JS. Никаких сборщиков и зависимостей — один файл `index.html`.

## Развёртывание (GitHub Pages)

1. Сделайте форк или клонируйте репозиторий
2. Перейдите в **Settings → Pages**
3. Source: **Deploy from a branch**, ветка `main`, папка `/ (root)`
4. Сайт будет доступен по адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`

## Настройка

Для смены логина/пароля найдите в `index.html`:
```js
const CREDS = { login: 'admin', password: 'Atremare' };
```

## Подключение МойСклад

1. МойСклад → Настройки → Безопасность → Токены доступа
2. Создать токен, скопировать
3. Вставить в поле токена в приложении — сохранится в браузере автоматически
